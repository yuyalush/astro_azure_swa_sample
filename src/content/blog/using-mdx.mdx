このテンプレートには [@astrojs/mdx](https://docs.astro.build/en/guides/integrations-guide/mdx/) インテグレーションが `astro.config.mjs` に設定済みです。MDX を使わない場合は、設定ファイルからインテグレーションを削除することで無効化できます。

## なぜ MDX を使うのか？

MDX は JavaScript と JSX 構文をサポートする Markdown の拡張形式です。これにより、[Markdown コンテンツ内に JavaScript や UI コンポーネントを混在](https://docs.astro.build/en/guides/integrations-guide/mdx/#mdx-in-astro)させることができ、インタラクティブなグラフやアラートなどを実装することが可能です。

既存の MDX コンテンツがある場合、このインテグレーションを使えば Astro への移行がスムーズに行えます。

## 使用例

以下は MDX 内で UI コンポーネントをインポートして使用する例です。  
ブラウザでこのページを開くと、クリック可能なボタンが表示されます。

import HeaderLink from '../../components/HeaderLink.astro';

<HeaderLink href="#" onclick="alert('clicked!')">
	MDX に埋め込まれたコンポーネント
</HeaderLink>
